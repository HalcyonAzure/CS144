# CS144 计算机网络 2021

基于斯坦福2021年的CS144课程，完善了一个简单的TCP聊天室。其中主要逻辑的实现过程记录在了自己的博客中，并在下方列出。

该课程有完善的单元测试和文档，在做完了所以实验以后明显加深了自己对TCP协议的理解，对于握手、挥手以及路由和MAC地址都有了自己的理解。

今年（2023年）的新课程已经有更新了，不过2021年的实验报告和仓库还是自己做一个记录备份，也许以后还用得上。
